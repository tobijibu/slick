lazyLoad

更新日: 2018.09.21

遅延読み込みの方法を指定します。 ondemandまたは、progressiveを指定することができます。 ondemandはスライドするとすぐに画像を読み込みます。 progressiveはページが読み込まれた後に非表示の画像を読み込みます。

  • 初期値 : linear
  • type : string
$('.slider').slick({
  lazyLoad: 'linear',
  useCSS: false,
});

デモ

ondemandはスライドした時に画像を読み込みます。 progressiveはページ読み込みが完了した後に画像を読み込みます。
(画像の読み込み状況はdeveloperツール等で確認してください。)

ondemand

progressive

JavaScript
$(document).ready(function(){
  $('.slider1').slick({
    lazyLoad: 'ondemand',
  });
  $('.slider2').slick({
    lazyLoad: 'progressive',
  });
});
