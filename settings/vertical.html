vertical

作成日: 2018.09.25

スライドの方向を縦方向にします。

  • 初期値 : false
  • type : boolean
$('.slider').slick({
  vertical: false,
});

デモ

スライド方向を横方向にしています。 "有効化"ボタンを押すとスライドの方向を縦方向にします。

1

2

3

有効化
JavaScript
$(document).ready(function(){
  $('.slider').slick({
    vertical: false,
  });
  var vertical = false;
  $('.js-setoption').on('click', function() {
    vertical = !vertical;
    if (vertical === true) {
      $(this).text('無効化');
    } else {
      $(this).text('有効化');
    }
    $('.slider').slick('unslick');
    $('.slider').slick({
      vertical: vertical,
    });
  });
});
HTML
<div class="slider">
  <p>1</p>
  <p>2</p>
  <p>3</p>
</div>
<div class="buttons">
  <div class="button js-setoption">有効化</div>
</div>
